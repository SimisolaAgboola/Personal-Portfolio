Tic-Tac-Toe Game

React JS

This project is a simple tic-tac-toe game made with react js. A player wins when either x or o is consecutive horizontally, vertically or diagonally. The program checks for the winner by checking for 9 possible combinations of x and/or o. If one is found, the winner is announced and the game is reset.

Resume Parser

Python | Flask

This project takes in as input different formats of a resume in either pdf or docx extension, parses the resume and extracts and fills out name, education, skills, email and phone number in the appropriate boxes on the application. It mimics a job application autofill from resume functionality as seen on workday, for example.

Full Stack Shopping Website

Vue JS | Node JS | MongoDB

This project is a full stack shopping website that allows users create user accounts, add items to their cart & checkout. It involves front end development using Vue JS framework, back end development using Node JS, a noSQL database using MongoDB and connecting the front end to the backend using Axios library. It also implements authentication using firebase authentication.
