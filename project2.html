Full Stack Ecommerce Website

Tech Used

Vue.js, Node.js, Express.js, Google OAuth, MongoDB, Vercel

What I created

A full stack ecommerce website with a functional shopping cart where users can create accounts, login, and add items from the products page to their cart

Lessons learned

  • Vue.js syntax such as template & script, v-router & v-if, this was my first time using vue framework
  • GoogleOAuth methods including issigninwithemaillink, etc
  • Axios for asynchronous fetching and updating of the database
  • How to connect the frontend to the backend for a functional full stack web app

Ideas to make it better

  • A favorites page so users can favorite products and view them later
  • Add the same item to the cart more than once with the quantity added displayed
  • Make the website fully responsive
