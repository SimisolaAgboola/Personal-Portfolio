AI News Summarization App

Tech Used

ReactJS Slick Carousel Node.js Express.js Axios NewsAPI, Gpt-4o OpenAI Api

What I created

A web app that fetches and displays the latest news in sports, entertainment and politics and summarizes them using an AI summarizer. This is intended to give users quick talking points they can use in any social interaction.

Lessons learned

  • Asynchronous API calls in ReactJS
  • Using an AI summarization API requires functionality and memory trade offs
  • Error handling API call responses
  • Managing complex application state efficiently to handle category-specific articles
  • Handling nested callback functions and promises while fetching, filtering, and summarizing news articles
  • Managing large files in github that exceed the file limit using git LFS

Ideas to make it better

  • Further simplify the summary by specifying the who, what, where & when of every news article using AI
