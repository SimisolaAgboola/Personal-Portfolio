Tech Used

Python, Flask, Custom Named Entity Recognition (NER) Kaggle Dataset

What I created

A simple interface where users upload a valid resume file format and the web app parses it and returns results of name, contact info, education & experience. Inspired by tiktok's seamless application interface.

Lessons learned

  • Annotating data for large language model training
  • Loading and training a new spacy model
  • Validating a large json file's syntax
  • Building a full stack application with flask
  • Using AWS textract in code

Ideas to make it better

  • Extract resume extras including links, extra curricular avicites, etc
  • Use AI for parsing as this is the most acurate way to achieve seamless resume pasrsing to date
