Google-Chrome-Reading-Extension

Tech Used

JavaScript Google Chrome developer tools

What I created

A chrome extension that highlights the 1st letter of every word on a web page when a users clicks the enable focus mode button. This feature enhances readability and allows for faster comprehension. Inspire by an article I read about learning techniques

Lessons learned

  • Writing a manifest.json file for a chrome extension
  • Running background scripts in a webpage
  • Uploading, testing and debugging a Chrome extension

Ideas to make it better

  • Adding a highligh feature and a side bar on the active tab where users can see highlighted key points from their reading as auto-generated bulleted notes
